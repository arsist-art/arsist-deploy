# Arsist Deploy

One-click deploy configuration for [Arsist](https://arsist.art) - the self-hosted portfolio and store for artists.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://dashboard.render.com/blueprint/new?repo=https://github.com/arsist-art/arsist-deploy)

## What is this?

This repo contains the deployment configuration for Arsist. It pulls the latest Docker image from Docker Hub and deploys it to your hosting platform.

## How to deploy

1. Click the "Deploy to Render" button above
2. Sign up or log in to Render
3. Enter your `LICENSE_KEY` (from your purchase email)
4. Set a secure `ADMIN_PASSWORD`
5. Click Deploy

Your gallery will be live in about 2 minutes!
